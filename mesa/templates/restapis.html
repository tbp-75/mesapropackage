REST API CATALOG


API works with a sample data set simulating products in an inventory.
            We can use each endpoint to manipulate this records and test each of them.
            

Getproducts: to retrieve current products in catalog.
                Method: POST
                URI: /api/Getproducts

                Returns json structure:

                {
                    "products": [
                      {
                        "name": "gold",
                        "price per gm": 60,
                        "quantity": 700
                      },
                      {
                        "name": "silver",
                        "price per gm": 23,
                        "quantity": 1500
                      }
                    ]
                  }

            

Addproducts: add new products to catalog. It takes a json structure as follows:
                Method: POST
                URI: /api/Addproducts
                Header: Content-Type : application/json
                Body: JSON

                {
                    "name": "lead",
                    "price per gm": 0.002,
                    "quantity": 2000
                  }
                Returns "Product added sucessfully" if transaction is OK.
            

Deleteproducts: remove products to catalog.
