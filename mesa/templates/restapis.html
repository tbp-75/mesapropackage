REST API CATALOG


API works with a sample data set simulating products in an inventory.
We can use each endpoint to manipulate this records and test each of them.
General call parameters are:

Method: POST
Header: Content-Type : application/json
Body: JSON


Getproducts: List current products in catalog. 

                        URI: /api/Getproducts
                        Returns json structure:
                {
                    "products": [
                      {
                        "name": "gold",
                        "price per gm": 60,
                        "quantity": 700
                      },
                      {
                        "name": "silver",
                        "price per gm": 23,
                        "quantity": 1500
                      }
                    ]
                  }


Addproducts: Add new products to catalog. It takes a json structure as follows: 

                        URI: /api/Addproducts

                        {
                            "name": "lead",
                            "price per gm": 0.002,
                            "quantity": 2000
                        }
                    Returns "Product added sucessfully" if transaction is OK.


Deleteproducts: Remove products from catalog. 

                        URI: /api/Deleteproducts
                        Simply pass the name of the article to delete:
                        {
                            "name":"copper"
                        }


Edit Products: Edit existing product in catalog. 


                        URI: /api/Editproducts
                        Pass the article as you wish it to be uptaded:

                        {
                            "name":"lead",
                            "price per gm":0.04,
                            "quantity":1200
                        }
